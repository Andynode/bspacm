Detailed Description

Include file to define UART instances for TM4C devices.

In your periph_config.c file you should define parameter macros described below, and include this file, once for each UART device you need in your application. Make sure the periph_config.c file also defines a pin mux assignment for the peripherals you will used.

Define:

  • BSPACM_INC_PERIPHNUM to the instance number that should be adjoined to the peripheral name (e.g. 0 for UART0, 1 for UART1).
  • BSPACM_INC_TX_BUFFER_SIZE (optional) to the desired size for the internal transmit fifo
  • BSPACM_INC_RX_BUFFER_SIZE (optional) to the desired size for the internal receive fifo

This will add the following symbols (where # denotes the peripheral number):

  • UART::_IRQHandler which gets installed in the interrupt vector
  • xBSPACMdeviceTM4CperiphUART# which may be used as a handle to operations described in <bspacm/periph/uart.h>.
